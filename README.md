Free Games Collection

A modern, responsive gaming website designed using Canva to showcase free-to-play games with an attractive cyberpunk-inspired UI.

Live Website

https://rizzusam.my.canva.site/free-games

About

Free Games Collection is a modern gaming landing page that helps users discover popular free-to-play games across different genres. The website features a sleek dark theme with vibrant purple accents and a responsive design that delivers an engaging experience on all devices.

Features

- Hero Banner
- Popular Free Games
- Game Categories
- Top Rated Games
- New Releases
- Why Play Free Section
- Responsive Layout
- Newsletter Subscription
- Search Bar
- Login Interface
- Footer with Quick Links and Social Media

Built With

- Canva Website Builder
- Modern UI/UX Design Principles
